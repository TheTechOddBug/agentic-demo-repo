# Interlude: Generalization Theory (VC-dimension)

**Goal:** understand why learning works at all. This is the most mathematical, least code-heavy topic in the course, so treat it as understanding-only. There is no lab here.

**Time:** 2 to 3 evenings.

## Train (watch and read)

1. **Generalization and VC-dimension** — Audit the corresponding 6.86x lecture.
2. **The clearest treatment anywhere** — Caltech, Learning From Data by Yaser Abu-Mostafa (free), the VC-dimension lecture: https://work.caltech.edu/telecourse.html
3. **Reinforce** — the bias-variance sections of ISLP: https://www.statlearning.com/

## Checkpoint

You can explain in plain words why a more complex model needs more data, and what VC-dimension is trying to measure. No implementation required.
