# Closing Topic: Natural Language Processing (and the bridge to modern LLMs)

**Goal:** point everything you built at text, and connect this course to the frontier of large language models. This is the natural on-ramp from 6.86x into the LLM and agent world.

**Time:** 1 week, plus an optional deeper build.

## Train (watch and read)

1. **Word embeddings** — Audit the 6.86x NLP application lecture. Then read Jay Alammar, The Illustrated Word2vec (the clearest visual explanation of representing words as vectors): https://jalammar.github.io/illustrated-word2vec/
2. **Transformers** — Jay Alammar, The Illustrated Transformer: https://jalammar.github.io/illustrated-transformer/

## Optional build (the best next step after this whole program)

Finish Andrej Karpathy's Zero to Hero series with "Let's build GPT from scratch": https://karpathy.ai/zero-to-hero.html

You already built backprop by hand in Unit 3. This extends that exact skill straight into a working transformer language model. It is the single best thing to do after 6.86x, and it lands you squarely in the LLM and agent space.

## Checkpoint

You can explain why embeddings capture meaning that bag-of-words cannot, and you have a working intuition for how the neural tools from Unit 3 scale up to language and modern LLMs.
